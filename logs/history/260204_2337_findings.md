# Technical Finding / Milestone

**Context**: Reglas de metales y microbiologicos

## The Discovery

Se ajusto el modelo para que metales pesados y microbiologicos se soliciten
por grupo: al activar el check, se incluyen todos los elementos/organismos.
Esto impacta el formato de solicitud, la estructura de detalle y la guia
de uso (sidebar/checklist).
